## Goals :
- Update to SDL3
- Resturcture to use CMake

## Inputs :
- **Sand :** A
- **Water :** Z
- **Dirt :** E
- **Fire :** R
- **Ice :** S
- **Erase :** D
- **Clear :** C
- **Big Brush :** B
- **Small Brush :** M
