 # SimFinitia
falling sand simulation

SDL project with C++ to make a falling sand Cellular automaton.

## Goals :
- Update to SD3
- Resturcture to use CMake

## Inputs :
- **Sand :** A
- **Water :** Z
- **Dirt :** E
- **Fire :** R
- **Ice :** S
- **Erase :** D
- **Clear :** C
- **Big Brush :** B
- **Small Brush :** M
